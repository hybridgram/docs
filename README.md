# Starlight docs (локальный фрагмент)

Эта папка содержит фрагменты документации для [hybridgram/docs](https://github.com/hybridgram/docs). Она создана для документирования изменений из PR и может быть объединена с полным клоном репозитория docs.

**Добавлено по PR #5 (Add aliases to polling command):**

- `src/content/docs/guides/polling.mdx` — страница про команду `hybridgram:polling` и её опции (включая короткие алиасы `-L`, `-F`, `-R`, `-W`, `-I`).
- `src/content/docs/changelog.md` — запись в changelog по PR #5.

Если у вас уже есть полный клон docs (Starlight), скопируйте эти файлы в соответствующие пути и при необходимости добавьте пункты в навигацию (`astro.config.mjs` / `astro.config.ts`).
